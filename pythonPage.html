Project I-DO

Project I-DO is a project I created along with two other students during my first semester at college in a python programming course. Project I-DO is a chatbot with additional functionality to it including the ability to tweet, play a random tune, and play a game. I learned how to use the piglet module to create the game portion of this project and contributed to the audio, twitter, and tkinter portion of the project.
